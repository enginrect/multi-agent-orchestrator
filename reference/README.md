# Reference Material

Files in this directory were copied from the `workload-cluster-add-on` repository
for reference only. They are **not part of the orchestrator application**.

These files illustrate the kind of repository and workflow that the orchestrator
is designed to manage.

## Contents

- `bootstrap/` — Kubernetes userdata templates (cp-init, cp-join, wk-join)
- `scripts/` — Cluster registration script for ArgoCD
- `docs/` — Architecture, onboarding, and service documentation from the target repo
- `docs/ai/` — Original agent role definitions and workflow spec
